## AirPods 设置

### Windows 下设置

1. 打开 AirPods 的外壳；
2. 双击正面绿灯下的为止，会发现从绿灯变成白灯；
3. 从 Windows 设置下选择 Add Bluetooth 设备，就可以在 Windows 下添加 AirPods 了；

### 在 Windows 下切换 AirPods 连接

1. 选择 Windows 桌面右下角的声音图标；
2. 在 Bluetooth 下选择待连接的 AirPods，就可以正常连接了；

### 在 iPhone 下 切换 AirPods 连接

1. 从 iPhone 的控制中心选择声音的图标；
2. 选择待连接的 AirPods，就可以正常连接了；
