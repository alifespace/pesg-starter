## Preparation

1. Register a Gmail Account
2. Install nodejs
   - `brew install nvm`
3. Install VSCode
4. Install VSC plug-ins
   1. Google Code Assist
   2. Live Server
   3. ES6 Code Snippets
5. Install Gemini-CLI
   - `npm install -g @google/gemini-cli`
6. Markdown Syntax

### VSC shortcut Key

1. CTRL+B: Left side pane
2. CTRL+1: Editor
3. CTRL+`: Terminal
4. ALT+G: Google Code Assist Chat
5. CTRL+K, V: Markdown Preview
6. CTRL+W: Close Current Window
7. CTRL+S: Save Current File
