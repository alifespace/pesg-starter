## Markdown

| 中文名称 (Chinese Term) | 英文名称 (English Term) | Markdown 语法 (Syntax Example)   |
| :---------------------- | :---------------------- | :------------------------------- |
| 标题                    | Heading                 | `# Heading 1` or ## `Heading 2`  |
| 粗体                    | Bold                    | `**Bold Text**`                  |
| 斜体                    | Italic                  | `*Italic Text*`                  |
| 粗斜体                  | Bold Italic             | `***Bold Italic Text***`         |
| 无序列表                | Unordered List          | `- Item 1` or `- Item 2`         |
| 有序列表                | Ordered List            | `1. Item 1` or `2. Item 2`       |
| 链接                    | Link                    | `[Link Text](URL)`               |
| 图片                    | Image                   | `![Alt Text](URL)`               |
| 行内代码                | Inline Code             | `const x = 1;`                   |
| 引用块                  | Block Quote             | `> This is a block quote.`       |
| 分割线                  | Horizontal Rule         | `---` or `***` or `___`          |
| 删除线                  | Strikethrough           | `~~Strikethrough Text~~`         |
| 任务列表                | Task List               | `- [ ] Task 1` or `- [x] Task 2` |

## Split Preview (Real-time Side-by-Side View)

If you want to open the preview next to the editing windows in VSC, allowing you to see the real-time rendering effect as you edit, the shortcut is:

- Windows/Linx: _Ctrl+K_ then _V_
- MacOS: _Cmd+K_ then _V_
