# vue 简介

## 渐进式框架

1. 渐进式意味着可以将 Vue 作为应用的一部分嵌入，带来更丰富的交互体验；
2. 如果希望把更多的业务逻辑使用 Vue，可以逐步使用 Vue，对原先项目逐步进行重构；
3. Vue 全家桶 Core+Vue-router+Vuex，可以满足各种需求；

## Vue 有很多特点和 Web 开发中的高级功能

1. 解耦视图和数据
2. 可复用的组件
3. 前端路由技术
4. 状态管理
5. 虚拟 DOM
